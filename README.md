# Diva

A cross-platform media player with social features, built with Kotlin Multiplatform and Jetpack Compose.

## Features

- **Media Playback**: Play audio and video files with a modern, intuitive interface
- **Library Management**: Organize your music and video collections
- **Social Features**: Share playlists, see what friends are listening to, and discover new content
- **Cross-Platform**: Available on Android, iOS, Desktop, and Web

## Tech Stack

- Kotlin Multiplatform
- Jetpack Compose
- Media3/ExoPlayer
